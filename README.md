# Flameds
Minitool that clears desktop from screenshots made by Flameshot by creating folder with data template name and moving all files to it.
# Installation
```
npm i -g flameds
```
# Usage
```
ds
```
## Configuration
### FROM
```
[$HOME]|[%USERPROFILE%]/Desktop
```
### TO
```
[$HOME]|[%USERPROFILE%]/Pictures/Flameshot
```
### Template
```
[{day}/{month}/{year}]
```
